# 项目相关repo

## 项目综述
本项目主要包含一个微信小程序, 及其后台配套系统.

本项目包含服务如下:
* Postgres数据库服务
* Redis服务
* API服务
* 后台管理服务(待开发)
* nginx服务
* elastic搜索服务(待开发)
* 队列服务(待开发)

全部服务使用docker管理.
